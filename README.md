## 📖 Sobre o Projeto

Este projeto apresenta minha trajetória pessoal com videogames ao longo da vida — desde a infância até o momento atual. O objetivo é mostrar de forma organizada os consoles que tive, os jogos que mais me marcaram em cada fase e como essas experiências influenciaram meu desenvolvimento pessoal.

Além do relato pessoal, o projeto conta com uma estrutura técnica completa, incluindo website, API, banco de dados e dashboard web.

---

## 🎯 Objetivos

- Apresentar os consoles e jogos que fizeram parte da minha vida
- Refletir sobre a evolução da minha relação com os games ao longo do tempo
- Desenvolver uma aplicação web completa como entregável técnico

---

## 🚀 Funcionalidades

- **Website** — apresentação da trajetória com consoles e jogos
- **API** — recebimento, tratamento e inserção de dados no banco
- **Banco de Dados** — armazenamento estruturado dos dados coletados
- **Dashboard Web** — visualização dos dados do banco de dados
